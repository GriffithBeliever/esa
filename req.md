# Event Scheduler Application


## Minimum Features:

 - Event Management: Add, edit, and delete events (title, date/time, location, and description).

 - Status Tracking: Mark events as "upcoming," "attending," "maybe," or "declined."

 - Implement any AI features you can think of.

 - Search: Find events by title, date range, location, or other criteria.


## Requirements:

 - Complete a working product in 1 hour using any language, framework, or development tool (including AI tools).

 - Provide your source code and a brief README explaining how to run it.

 - Deploy the app and provide a URL for live testing.

 - Add user accounts with invitation capabilities.

 - Add extra features as you see fit, to demonstrate your creativity.
